# CIX PPA Documentation

This site publishes the official CIX PPA user manual as a bilingual documentation set built with MkDocs and deployed through GitHub Pages.

[English Documentation](en/index.md){ .md-button .md-button--primary }
[中文文档](zh/index.md){ .md-button }

## Quick Start

```bash
sudo apt install curl
curl -fsSL https://archive.cixtech.com/cix-repo-community.sh | sudo sh
```

## What You Will Find Here

- Installation guidance for Debian 13 GNOME on CIX ARM64 platforms
- Repository setup instructions for `cix-repo-community.sh`
- Driver package details, maintenance commands, and troubleshooting notes
- Mirrored English and Simplified Chinese documentation trees
